A Data-Acquisition Robot for Acoustic Sensing & Localization (Robot Audition)

Developed a robot audition platform

Work done with Dr. Sumohana Chhannappayya (IIT Hyderabad) and Dr. K Sri Rama Murty (IIT Hyderabad)

(July 2021 - May 2022)

  • Implemented robust and precise sound source localization and tracking using an 8-channel microphone array on an all-terrain robot developed by myself. (The height of the microphone array is adjustable up to 1.7m).
  • Developed a communication system using the 4G protocol, with an onboard HDD for live data storage streaming from state estimation and other sensors.
