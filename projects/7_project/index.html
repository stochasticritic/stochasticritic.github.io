DDP-based solver for various Optimal Control problems under contact sequence

Course project for M270C: Optimal Control

  • Implemented BOX-FDDP for various dynamical systems using the Multi-Contact Optimal Control Problem (MCOP) formulation.
  • The cost function includes waypoints as desired poses and velocities:
    J = ∑(t=0 to T) [ xᵗQx + uᵗRu ]
    where Q and R are weight matrices for state xt and control ut.
  • Control constraints are enforced via a feasibility-driven approach, modifying the standard DDP update as:
    δut = -Kt δxt + kt
    where Kt is the feedback gain and kt is the feedforward term computed under relaxed feasibility constraints.
  • Early feasibility projection improves stability, ensuring feasible control updates within the constraint set:
    utnew = Π[umin, umax](ut + δut)
    where Π is the projection operator onto the control limits.
  • Compared to Box-DDP, this approach reduces numerical instability and produces smoother trajectories in quadruped locomotion and humanoid manipulation.
