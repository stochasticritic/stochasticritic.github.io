Word done with [Dr. Sumohana Chhannappayya](https://people.iith.ac.in/sumohana/) (IIT Hyderabad) and [Dr. K Sri Rama Murty](https://sites.google.com/iith.ac.in/ksrm/home?authuser=0) (IIT Hyderabad)

(July 2021 - May 2022) 

• Implemented robust and precise sound source localization and tracking (using an 8-channel microphone array) on an all-terrain robot
developed by myself. (The height of the microphone array is adjustable up to 1.7m)

• Communication is using the 4G protocol, and an HDD is present onboard for live data storage, which is streaming from the state
estimation and other sensors.

Follow <a href="https://github.com/stochasticritic/Audio-Visual-Data-Acquisition-Platform" >this repository</a> for more details.
